# Bookmark M

面向移动端触控体验的书签管理扩展，支持快速浏览、搜索、整理、移动、编辑与批量操作。

## 功能

- 适合手机、平板和触屏设备的书签管理界面
- 支持列表模式与树形模式切换
- 支持搜索书签和文件夹
- 支持新建、编辑、移动、删除书签与文件夹
- 支持批量选择、批量打开与顺序整理
- 支持将当前页面快速加入书签

## 技术栈

- WXT
- React 19
- TypeScript
- Tailwind CSS

## 开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
pnpm build:zip
```
